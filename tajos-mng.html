☁ ¿Cómo funciona el almacenamiento?

Los datos se guardan en Supabase (nube) como fuente principal. El navegador mantiene una copia local en IndexedDB como caché y fallback offline. Cuando la conexión vuelve, las operaciones pendientes se sincronizan automáticamente.

⚠ Consideraciones
• Sin conexión a internet, los datos se guardan localmente y se sincronizan al reconectar
• La caché local se actualiza automáticamente al abrir cada sección
• Si limpiás la caché del navegador, los datos en Supabase no se pierden
✅ Buenas prácticas
• Aún se recomienda exportar un backup al final de cada jornada como respaldo adicional
• Usa JSON para uso general, XML si necesitas abrir en Excel
• El indicador verde/rojo en la esquina inferior derecha muestra el estado de conexión con Supabase
