# COMPLETE LANDING PAGE SPECIFICATION FOR TAJOS BARBEROS
## Comprehensive Visual and Functional Description

---

## OVERVIEW

This is a multi-page website for "TAJOS BARBEROS," a premium barbershop brand with locations in Paraguay. The website serves as a landing page for potential clients to learn about the brand, view services, explore locations, and book appointments. The design emphasizes a sophisticated, elegant aesthetic with warm, professional lighting.

---

## GLOBAL DESIGN SYSTEM

### Color Palette
**Primary Color (Accent/Highlight):** #fdd835 (Warm yellow/gold)
- Used for: Logo text, headings, buttons, highlights, hover states, icons, active navigation states

**Background Color:** #0d1421 (Deep dark blue, almost black)
- Main page background with an overlay of warm office lighting photograph
- Creates an elegant, premium atmosphere

**Card Background:** #1a2332 (Slightly lighter dark blue)
- Used for cards, navigation bar, location cards, testimonial cards, and dialog backgrounds
- Has subtle transparency (80-95%) with backdrop blur for glassmorphic effect

**Text Colors:**
- Primary headings: #fdd835 (yellow/gold)
- Body text: #ffffff (white)
- Secondary/muted text: #e0e0e0 to #808080 (light gray to medium gray)

**Border Colors:**
- rgba(255, 221, 53, 0.2) - semi-transparent yellow borders
- Subtle borders that complement the dark theme

### Typography
**Font Size Base:** 16px (1rem)

**Heading Hierarchy:**
- H1: Very large (6xl to 8xl on mobile to desktop), thin weight, wide letter spacing, all caps for logo
- H2: 4xl, medium weight, used for section titles
- H3: xl to 2xl, medium weight, used for card titles and subsection headings
- Body text: Base size (16px), normal weight, line height 1.5

**Font Styling:**
- Logo: Thin (font-weight 100), extremely wide tracking, uppercase
- Buttons: Medium weight
- All text: Smooth rendering, good readability against dark backgrounds

### Background Treatment
- Fixed background image: Warm office environment with ambient lighting (desk lamp, soft indirect lighting)
- Image URL reference: Dark office with warm lighting and lamps aesthetic
- Semi-transparent dark blue overlay (rgba(13, 20, 33, 0.85)) over the background image
- Creates depth and ensures text readability
- Background is fixed (doesn't scroll with content)

### Spacing & Layout
- Sections: 16px vertical padding (py-16)
- Containers: Max width of 6xl (72rem/1152px) centered
- Gap between elements: Typically 4-8px for tight groupings, 12-24px for related content
- Rounded corners: Generally 8-10px (0.625rem) for cards and buttons

### Interactive Elements
**Buttons:**
- Primary button: Yellow background (#fdd835), dark blue text (#0d1421)
- Outline button: Transparent background, yellow border, white/yellow text
- Rounded corners (lg radius)
- Hover: Slightly darker/lighter shade (90% opacity)
- Smooth transitions (300ms duration)

**Hover States:**
- Images: Scale 105% on hover with smooth transition
- Cards: Border changes from semi-transparent to visible yellow
- Buttons: Background opacity changes
- Links: Text color changes to yellow
- All transitions: 300ms ease

**Navigation Active States:**
- Active section: Yellow background with dark blue text
- Inactive: White text with transparent background
- Hover: Yellow text with light yellow background (10% opacity)

## RESPONSIVE BEHAVIOR

### Breakpoints:
- Mobile: < 768px (md)
- Tablet: 768px - 1024px (md to lg)
- Desktop: >= 1024px (lg)

### Navigation:
- Desktop: Full 5-item navigation visible
- Mobile: Only 3 items visible + menu button

### Grid Layouts:
- Testimonials: 1 col (mobile) → 3 cols (desktop)
- Locations: 1 col (mobile) → 2 cols (md) → 3 cols (lg)
- Events: 1 col (mobile) → 2 cols (md) → 3 cols (lg)
- Gallery carousel: 1 item (mobile) → 2 items (md) → 3 items (lg)

### Text Sizes:
- Headings scale down on mobile (6xl → 4xl for hero)
- Body text remains readable at base size
- Buttons maintain minimum touch target size (44px)

### Spacing:
- Padding reduces slightly on mobile
- Gaps between elements reduce proportionally
- Maintains visual hierarchy across all sizes

### Images:
- Maintain aspect ratios
- Use object-cover for consistency
- Load appropriate sizes for device
- Always include alt text for accessibility
