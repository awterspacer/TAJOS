# MENU PAGES COMPLETE SPECIFICATION FOR TAJOS BARBEROS
## Detailed Description of All Alternative Pages from Navigation Menu

---

## OVERVIEW

This document describes all the pages accessible from the dropdown navigation menu (the "pill menu" with hamburger icon) on the TAJOS BARBEROS website. Each page shares the same design system as the main landing page but serves different informational purposes. All pages are in Spanish.

---

## GLOBAL PAGE STRUCTURE

All menu pages share these common elements:

### Header Section
**Back Button:**
- Position: Top-left corner
- Style: Outline button with icon only
- Icon: Arrow left (pointing left)
- Size: Icon size (square button)
- Border: Subtle yellow border
- Hover: Light yellow background
- Action: Returns to home page, scrolls to top

**Page Title Area:**
- Adjacent to back button (12px gap)
- Main title: 3xl size, yellow color (#fdd835), margin bottom 8px
- Subtitle: Gray color (#d1d5db), normal size
- Layout: Vertical stack

### Container
- Full viewport height minimum
- Dark background (#0d1421) with same overlay as home
- Container: Max width, centered, padding 16px horizontal, 32px vertical
- Scrollable content

### Design Consistency
- Same color system as landing page
- Same typography
- Same card styling (dark cards with subtle borders)
- Same button styles
- Same hover effects and transitions

## MENU OPTION 1: CONTÁCTANOS (Contact Us)

**Note:** This is NOT a separate page. It's a direct action button.

**Behavior:**
- Opens WhatsApp in a new browser tab
- Phone number: 1234567890 (format: +1234567890)
- Pre-filled message: "Hola, me gustaría agendar una cita en TAJOS BARBEROS"
- URL format: https://wa.me/1234567890?text=[encoded message]

**Visual Indicator:**
- Menu item has message circle icon (16px)
- White text
- Hover: Yellow text with light yellow background
